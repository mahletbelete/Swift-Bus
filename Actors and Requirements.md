Swift Bus Ticketing System (SBTS)
The swift Bus Ticketing System (SBTS) is a web-based application designed to digitalize and automate the bus ticketing process.
It enables passengers to search for bus routes, check schedules, select seats, make online payments, and receive electronic tickets with QR codes.
The system also supports bus operators and administrators by providing tools to manage buses, routes, schedules, fares, users, and financial reports.
Overall, SBTS replaces manual ticketing with a secure, efficient, and user-friendly digital solution.

3.As a passenger, I want to search for buses by route and date so that I can plan my trip.
4.As a passenger, I want to view seat availability so that I can choose my preferred seat.
5.As a passenger, I want to book a seat so that I can reserve my travel.
6.As a passenger, I want to pay online using Telebirr or CBE Birr so that booking is convenient.
7.As a passenger, I want to receive a digital ticket with a QR code so that I can travel without paper.
8.As a passenger, I want to view my booking history so that I can manage past and future trips.
